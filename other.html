Botanical Maestro
Using a webcam and through hand detection, a user is able to control the position for randomly generated flowers or clear the screen. Try it here. Developed using p5.js and handtrack.js, based on Lindenmayer systems.
Screaming Game
A very simple game with a focus on sound: the player's character, multiple enemies; when the player shouts, the enemies die. If the sound is loud enough, the player gets a x2, or even x3, bonus. Additionally, all background music is randomly generated, when the tab is opened. Try it here. Developed using p5.js and p5.play, inspired by games such as Scream Go Hero and Flappy Scream Chicken.
